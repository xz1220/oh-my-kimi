# Remember

Use this when a decision, convention, or lesson should be recorded.

## Rules

- Record only facts from explicit user input, source material, or observed code.
- Put the note in the smallest appropriate durable file.
- Include date and context when the note affects future decisions.
- Mark uncertainty instead of filling gaps.
- Do not store secrets, personal data, or financial facts unless explicitly
  directed and the repository is the right place.
