# Deep Dive

Use this when a shallow answer is likely to miss important context.

## Workflow

1. Define the question and expected decision or artifact.
2. Map likely information sources.
3. Search broadly, then read narrowly.
4. Separate observed facts, inferences, and open questions.
5. Cross-check against at least one independent source or code path when the
   conclusion matters.
6. Produce a compact synthesis with evidence references.

## Output

- Question
- Findings
- Evidence
- Risks / uncertainty
- Recommended next action
