# Autopilot

Use this when the user wants a complete outcome and has delegated execution.

## Workflow

1. If the requirement is vague, run the deep-interview pattern first.
2. Produce a compact ralplan-style story breakdown.
3. Execute story by story using the Ralph loop.
4. Delegate bounded work to subagents when it reduces risk or latency.
5. Maintain a visible task list.
6. Run full relevant validation before final response.
7. Summarize changed files, tests, and residual risks.

## Escalation

Ask the user only when progress is blocked by an irreversible decision,
missing secret, financial/legal fact, or destructive operation.
