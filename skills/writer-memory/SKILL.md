# Writer Memory

Use this when a writing convention should influence future docs.

## Guidelines

- Capture concrete preferences, not vague taste.
- Include examples when useful.
- Keep notes project-scoped unless they are broadly reusable.
- Do not overwrite existing style guidance without checking the surrounding
  docs.

For a one-off writing task, follow the existing document's voice instead of
creating new memory.
