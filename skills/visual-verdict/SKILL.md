# Visual Verdict

Use this for frontend or documentation pages where layout quality matters.

## Workflow

1. Start or identify the local server.
2. Capture desktop and mobile screenshots.
3. Check text overflow, overlap, spacing, contrast, loading states, and empty
   states.
4. Exercise the primary interaction path.
5. Fix visible issues and recapture screenshots.

A visual pass requires direct observation, not just successful compilation.
