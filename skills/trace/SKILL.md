# Trace

Use this for evidence-based causal tracing.

## Workflow

1. Start from an observable symptom or entry point.
2. Identify upstream inputs and downstream effects.
3. Follow actual call paths, config paths, events, or data transformations.
4. Track competing hypotheses and evidence for or against each.
5. Stop when the next action is clear: fix, test, instrument, or escalate.

## Output

- Trace path
- Evidence
- Hypotheses eliminated
- Most likely cause
- Next probe or fix
