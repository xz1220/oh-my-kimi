# oh-my-kimi

Kimi CLI 的精选 skill / hook / MCP 包，装上就有 oh-my-claudecode 风格的 agent team、经典 hook 和常用 MCP 一键配齐。

灵感源于 [oh-my-claudecode](https://github.com/Yeachan-Heo/oh-my-claudecode) 和 [oh-my-codex](https://github.com/Yeachan-Heo/oh-my-codex)。Kimi CLI 的扩展面（Skill / Hook / MCP / Subagent / AGENTS.md）和 Claude Code 同构度极高，且原生读 `~/.claude/skills/` —— 这意味着同一份 skill 包**对 Kimi CLI 和 Claude Code 都生效**。

详细需求看 [docs/prd.md](docs/prd.md)。
